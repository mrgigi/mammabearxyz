# mammabear

A cozy, neon-gradient meme site for `$MAMMABEAR`.

## Preview locally

```
python3 -m http.server 5500
# open http://localhost:5500/
```

## Deploy

### 1) GitHub
- Create a repo named `mammabearxyz` (or your preferred name).
- Push this directory to GitHub.

### 2) Vercel
- Connect your GitHub repo in the Vercel dashboard.
- Project type: Static Site.
- Root directory: `/`.
- Build/Output: none needed; Vercel serves static files automatically.

Optional config lives in `vercel.json`.
